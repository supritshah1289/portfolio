Deal effect

Daily deals

Deal-effect is a website developed using HTML, CSS, JavaScript, MongoDB, NodeJS, jQuery, Ajax, API. Thanks to sqoot for providing trial API that have daily deals of more than 500k. Hosted on Heroku.

Deal effect shows approximately 50 daily deal to guest user. Guest can sign up and become users for more benefits. Our users can save daily deals to their account. Users can also delete saved deals from their account. Users can search online deals as well as stores near by. Users can view deals based on page number of Sqoot API, Also they can limit number of deals per page for better experience.

You can preview Deal effecthere.

Staycation

Staycate and enjoy the adventures locally

Staycation is your one stop place to help you figure out what you can do in whatever zip code you'd like to visit; within whatever budget you set.

The project was developed as a team environment. We began with brainstorming Idea on what we will be building as a team. Then we wireframe our single page application. As a group we invested our time for searching best API that will fit our requirement of having place to stay and things to do around the place. We came across awesomely documented API from Expedia that has two end points those are things to do API and Hotels api.

Once we had our building block ready. As a group we had analyze part of the project and created sprints for each single scrums to be completed for each single day. We have created trello board for tracking our day to day activities.

I have worked on both the sides of the project. At times I was playing role on front-end and back-end as well. I have completed all task that I was assigned too. Sometimes I require help from back-end developer on what particular request he is expecting to respond back with data. Also, When I get stuck on some of the React part I seek my team members help as well as their suggestions.

Overall results of building Single Page Application in team was satisfying, However, we were not able to complete authentication system for the application because of the complexity of React token exchanging. Time was also an important factor of this project because we only had 4 days.

Working in a team environment I have learned lot. We have utilized trello boards to keep track of our project, team meeting every morning.We have set up deadline for each day to accomplish working functionality of our product. I was also involved in pair programming with team member.

Please visit us staycation.
